# Clip Video

Intelligent video clip extraction tool for lyric matching and highlights generation.

## Installation

```bash
pip install -e .
```

## Usage

```bash
clip-video --help
```

## Modes

### Lyric Match Mode
Build searchable word/phrase dictionaries from video libraries for creating music video mashups.

### Highlights Mode
Batch-process recordings to generate social media shorts with burned-in captions.
